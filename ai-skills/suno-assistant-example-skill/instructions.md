# suno-assistant Example Skill

Use this directory as the smallest starting point for a new canonical suno-assistant skill.

Workflow:
1. Rename the directory and update `skill.yaml`.
2. Replace this file with the actual instructions the skill should follow.
3. Add optional `references/`, `scripts/`, or `assets/` directories when the skill needs them.
4. Run `./scripts/deploy_ai_skills.sh` to render the skill to Claude and Codex.

Keep the body concise, action-oriented, and specific to the situations where the skill should be used.
